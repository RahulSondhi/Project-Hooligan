    Movie List Database


    Project description
    A movie watch list database was created using Java in which a user can make a list of all movies they have watched to keep track of it. User input a movie title into program which grabbed information about the movie from another movie database. Information that was included was date of release, actors, title of film and movie poster. Films were then sorted in database based on user preference (Release date, Title, or Actors in movie). Use of movie posters as icons, made the program aesthetically pleasing.


    Smart Watch


    Project description
